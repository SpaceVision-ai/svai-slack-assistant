# Space-Gemini 사용자 가이드

## 1. 개요

Space-Gemini는 Google의 강력한 AI 모델인 Gemini (gemini-2.5-pro, gemini-2.5-flash)를 기반으로 동작하는 지능형 Slack 봇입니다. 단순히 텍스트에만 응답하는 것을 넘어, 대화의 맥락을 이해하고 첨부된 파일이나 링크의 내용을 분석하여 종합적인 답변을 제공합니다.

## 2. 주요 기능

- **대화 맥락 이해**: 채널이나 스레드에서 오고 간 대화를 파악하여 질문의 배경을 이해하고 답변합니다.
- **기간 설정 기능**: "어제", "지난 3일", "이번 주" 등 특정 기간을 지정하여 해당 기간의 대화 내용만 참고하도록 요청할 수 있습니다.
- **다양한 파일 형식 지원**:
  - 이미지 및 동영상 (`image/*`, `video/*`)
  - PDF 문서 (`.pdf`)
  - MS Word 문서 (`.docx`)
  - 텍스트 파일 (`.txt`, 등)
- **웹 페이지 및 링크 분석**:
  - 일반 웹사이트 URL
  - Notion 페이지 링크
  - 다른 Slack 메시지 링크(Permalink)
- **자동 응답 길이 조절**: 생성된 답변이 Slack 메시지 길이 제한을 초과할 경우, 자동으로 텍스트 파일(`.md`)로 만들어 업로드합니다.

## 3. 사용 방법

### 기본 사용법

- **채널에서 호출**: 메시지 입력창에 `@Space-Gemini`를 입력하여 봇을 맨션하고 질문을 입력합니다.
- **개인 메시지(DM)**: 봇과의 DM 채널에서 바로 질문을 입력합니다.

### 파일 및 링크 활용

- **파일 질문**: 이미지, PDF, DOCX 등의 파일을 메시지와 함께 업로드하며 질문합니다.
  > `@Space-Gemini 이 문서의 내용을 요약해 줘.`
- **링크 질문**: 분석하고 싶은 웹 페이지, Notion 페이지, Slack 메시지 링크를 포함하여 질문합니다.
  > `@Space-Gemini 아래 노션 페이지의 핵심 내용을 알려줘. https://...`

### 대화 맥락 이해 기능 활용

- **자동 맥락 분석**: 별도의 지정이 없으면, 봇은 질문에 답변하기 위해 맥락이 필요한지 스스로 판단합니다. 맥락이 필요하다고 판단되면, 기본적으로 **최근 3일**의 대화 내용을 참고합니다.
- **기간 직접 지정**: 특정 기간의 대화 내용을 참고하게 하려면, 다음과 같이 기간을 명시하여 질문합니다.
  > `@Space-Gemini **어제 논의했던 내용**을 바탕으로 기획안 초안을 작성해 줘.`
  > `@Space-Gemini **지난 7일 동안** 공유된 링크들 목록을 만들어 줄래?`

## 4. 사용 예시

- **간단한 질문**
  > `@Space-Gemini 우리 프로젝트의 다음 마일스톤이 언제야?`

- **파일 기반 질문**
  > (회의록.docx 파일을 업로드하며) `@Space-Gemini 이 회의록에서 결정된 주요 액션 아이템들을 정리해 줘.`

- **링크 기반 질문**
  > `@Space-Gemini 이 블로그 글을 읽고 마케팅팀에 공유할 만한 인사이트를 뽑아줘. https://...`

- **맥락 기반 질문**
  > `@Space-Gemini **이번 주에** 나왔던 고객 피드백들을 종합해서 보고서 형식으로 만들어 줘.`
