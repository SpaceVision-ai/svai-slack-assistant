# Translate-Gem 사용자 가이드

Translate-Gem은 슬랙 채널과 다이렉트 메시지에서 실시간으로 메시지를 번역해주는 봇입니다. 한국어는 영어로, 그 외 모든 언어는 한국어로 자동 번역합니다.

## 1. 공개/비공개 채널에서 사용하기

채널에서는 특정 채널을 지정하여 실시간 번역을 활성화해야 합니다.

### 실시간 번역 활성화

번역을 원하는 채널에서 다음 명령어를 입력하세요.
```
/translate-gem-channel add
```
- 이제 해당 채널에 올라오는 모든 메시지(수정 포함)가 자동으로 번역됩니다.
- 번역된 내용은 원본 메시지의 스레드(댓글)로 달립니다.

### 실시간 번역 비활성화

번역을 중단하고 싶으면 언제든지 다음 명령어를 사용하세요.
```
/translate-gem-channel remove
```

### 번역 활성 채널 목록 보기

현재 번역이 활성화된 모든 채널의 목록을 보려면 다음 명령어를 사용하세요.
```
/translate-gem-channel list
```

## 2. 1:1 다이렉트 메시지 (DM)에서 사용하기

Translate-Gem 봇과의 1:1 대화방에서는 별도의 설정 없이 바로 사용할 수 있습니다.

- **사용 방법**: 그냥 메시지를 보내세요!
- 봇에게 보내는 모든 메시지(수정 포함)는 자동으로 번역되어 일반 메시지로 답장이 옵니다.

## 3. 여러 명이 있는 그룹 다이렉트 메시지 (그룹 DM)에서 사용하기

여러 명과 함께 있는 그룹 다이렉트 메시지에서도 별도의 설정 없이 자동으로 번역 기능이 활성화됩니다.

- **사용 방법**: 그룹 DM에 Translate-Gem 봇을 초대하기만 하면 됩니다.
- 봇이 초대되면, 자동으로 번역을 시작하겠다는 안내 메시지를 보냅니다.
- 이후 해당 대화방에 올라오는 모든 메시지(수정 포함)는 자동으로 번역됩니다.
- 번역된 내용은 누가 말했는지 알 수 있도록 원본 작성자를 멘션하며 일반 메시지로 전송됩니다.
