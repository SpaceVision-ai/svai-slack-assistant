# Translate-Gem 사용자 가이드

Translate-Gem은 슬랙 채널과 다이렉트 메시지에서 실시간으로 메시지를 번역해주는 봇입니다. 한국어는 영어로, 그 외 모든 언어는 한국어로 자동 번역하여 원활한 소통을 돕습니다.

## 1. 기본 번역 기능

- **자동 언어 감지 및 번역**: 한국어 메시지는 영어로, 그 외 모든 언어(영어, 일본어 등)는 한국어로 자동 번역됩니다.
- **번역 진행 안내**: 내부적으로 Google의 `gemini-2.5-flash` 모델을 사용하여 번역을 수행하므로, 수 초의 짧은 대기 시간이 발생할 수 있습니다. 번역이 진행되는 동안에는 잠시 기다려달라는 안내 메시지가 표시됩니다.
- **정확한 작성자 표시**: 번역된 메시지에는 항상 원본 메시지 작성자를 멘션하여 누가 말했는지 명확히 알 수 있습니다.
- **채널/DM 유형별 응답 방식**:
    - **공개/비공개 채널**: 번역문은 원본 메시지의 스레드(댓글)로 달립니다.
    - **1:1 및 그룹 DM**: 번역문은 해당 대화방에 일반 메시지로 전송됩니다.

## 2. 채널에서 사용하기

봇을 초대한 채널에서 특정 명령어를 통해 실시간 번역을 활성화 또는 비활성화할 수 있습니다.

- **번역 활성화**: ` /translate-gem-channel add `
  - 해당 채널의 모든 메시지가 실시간으로 번역됩니다.
- **번역 비활성화**: ` /translate-gem-channel remove `
  - 실시간 번역이 중단됩니다.
- **활성 채널 목록 보기**: ` /translate-gem-channel list `
  - 현재 번역이 활성화된 모든 채널 목록을 보여줍니다.

## 3. 다이렉트 메시지(DM)에서 사용하기

- **1:1 DM**: 봇과의 1:1 대화에서는 별도 설정 없이 모든 메시지가 자동으로 번역됩니다.
- **그룹 DM**: 여러 명과 함께 있는 그룹 DM에 봇을 초대하면, 그 즉시 자동 번역 기능이 활성화됩니다.

## 4. Notion 연동: 문서 제목 번역

사내 규칙에 따라 Notion 문서 제목을 영어로 표준화하는 과정을 돕습니다.

- **기능 설명**: 채널이나 DM에서 Notion 문서 링크를 공유했을 때, 해당 문서의 제목이 한글이면 봇이 자동으로 영어 번역을 제안합니다.
- **동작 방식**:
    1. 봇이 한글 제목을 감지하면, **`기존 한글 제목 (번역된 영어 제목)`** 형식으로 새로운 제목을 제안하는 버튼을 스레드에 표시합니다.
    2. **`✅ 네, 변경합니다`** 버튼을 누르면 해당 Notion 문서의 제목이 제안된 형식으로 즉시 업데이트됩니다.
    3. 원하지 않으면 **`아니요`** 버튼을 눌러 제안을 닫을 수 있습니다.
- **권한 오류 안내**: 만약 봇에게 Notion 페이지 접근 권한이 없다면, 다음과 같은 안내 메시지가 표시됩니다. 이 경우, 해당 Notion 페이지 상단의 **`공유(Share)`** 버튼을 눌러 **`Translate Gem`** 봇을 초대해주시면 정상적으로 작동합니다.
    > :warning: Notion 페이지에 접근할 수 없습니다. 페이지가 존재하지 않거나, 저에게 접근 권한이 없는 것 같아요.
    >
    > :warning: I can't access that Notion page. It might not exist, or I may not have permission.

## 5. 향후 계획

- **Notion 문서 전체 번역**: 현재는 제목 번역만 지원하지만, 앞으로는 Notion 문서 본문 전체를 번역하여 새로운 페이지를 생성하는 기능이 추가될 예정입니다.
