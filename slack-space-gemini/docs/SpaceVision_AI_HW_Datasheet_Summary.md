## 1. SpaceVision AI Edge Device

**모델명:** SpaceVision AI Edge Device 2.0

### 주요 특징 (Key Features)
- **AI 성능:** 최대 26 TOPS NPU 탑재
- **프로세서:** Quad-core Intel Alder Lake CPU (Max 3.6GHz)
- **메모리:** 8GB DDR4
- **전력 소모:** 42W (약 40W)
- **연결성:** 2x USB 3.2 등
- **크기:** 130 x 100 x 48 mm

### 기술 사양 (Technical Specifications)

| 항목 | 상세 내용 |
| :--- | :--- |
| **Processor** | Intel® Alder Lake (Quad Core, Max 3.6GHz) |
| **AI Accelerator** | 26 TOPS NPU |
| **Memory** | 8GB DDR4 |
| **Graphics** | Intel® UHD Graphics or ARM Mali-G52 |
| **Storage** | 256GB M.2 (PCIe x2/SATA) |
| **Video Output** | 1x HDMI 2.1, 1x DP 1.4 |
| **Ethernet** | 1x 2.5Gbps RJ45, 1x 1Gbps RJ45 (WOL Support) |
| **Wireless** | Built-in Wi-Fi & Bluetooth Module (M.2 Key E 2230) |
| **USB** | 2 x USB 3.2, 2 x USB 2.0 (Optional) |
| **Power** | DC 12V / 3.5A Adapter |
| **Dimensions** | 130 x 100 x 48 mm (W x D x H) |
| **Mounting** | VESA Mount (70x70 mm / 100x100 mm) |
| **Supported OS** | Windows IoT 10/11, Ubuntu 20.04 or later, Debian, Android (ARM Option) |
| **Software Support** | SpaceVision’s AI-powered Media & Analytics, Device Management & Monitoring System, Installation Guide Software |
| **Software Updates** | Over-the-Air (OTA) Support |

## 2. SpaceVision AI Sensor (Camera)

**모델명:** SpaceVision CSL100

### 주요 특징 (Key Features)
- **해상도:** 1920 x 1080 @ 30 fps
- **시야각 (FOV):** 120° (Diagonal)
- **크기:** 90 x 13 x 13 mm (브라켓 제외)
- **인터페이스:** USB 2.0

| 항목 | 상세 내용 |
| :--- | :--- |
| **Sensor** | 1/4" |
| **Active Array Size** | 1920 x 1080 |
| **Max. Frame Rate** | 30fps @ 1920x1080 (MJPG) |
| **Pixel Size** | 1.4µm x 1.4µm |
| **Lens Construction** | 4P |
| **F/No** | 2.0 |
| **FOV (D/H/V)** | 120° |
| **Output Formats** | MJPG, YUV |
| **Interface** | USB 2.0 |
| **Power** | USB Bus Power |
| **Dimensions** | 90 x 13 x 13 mm (W x D x H, w/o bracket) |
| **Operating Temperature** | -20°C to 70°C |
